Propiedade object-fit

object-fit é uma propriedade CSS que especifica como o conteúdo de um elemento substituído deve ser redimensionado para se ajustar ao contêiner que o contém.

object-fit: fill; (padrão) - O conteúdo é redimensionado para preencher o contêiner. O conteúdo pode ser distorcido para preencher o contêiner.


object-fit: contain; - O conteúdo é redimensionado para caber no contêiner, preservando sua proporção original.


object-fit: cover; - O conteúdo é redimensionado para preencher o contêiner, preservando sua proporção original. O conteúdo pode ser cortado para preencher o contêiner.


object-fit: none; - O conteúdo é redimensionado para caber no contêiner, preservando sua proporção original. O conteúdo não é redimensionado para preencher o contêiner, e pode sobrar espaço em branco.


object-fit: scale-donw; - O conteúdo é redimensionado para caber no contêiner, preservando sua proporção original. O conteúdo não é redimensionado para preencher o contêiner, e pode sobrar espaço em branco.
