Agrupamento de Seletores

Podemos aplicar as mesmas regras CSS para varios seletores diferentes. Para isso, podemos separar os seletores utilizando uma virgula.

Assim, não presisamos ficar copiando os mesmos codigos para todos, nós podemos utilizar apenas um bloco de CSS e reutilizar as mesmas regras.
